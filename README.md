# Gestion de Microfinance : Cas de COOPED 

LE BACKEND

## Description
Application web et mobile permettant de gérer les opérations d'une microfinance :
- Création de tontines
- Dépôts
- Retraits
- Prêts
- Paiements des frais
- Achats

## Technologies
- Flutter
- ASP.NET Core Web API
- MySQL

## Installation

1. Cloner le projet
```bash
git clone https://github.com/utilisateur/microfinance.git
```

2. Aller dans le dossier
```bash
cd microfinance
```

3. Installer les dépendances
```bash
flutter pub get
```

4. Lancer l'application
```bash
flutter run
```
